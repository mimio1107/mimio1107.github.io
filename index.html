Web Designer Miura Orie

東京都在住、Webデザイナー志望の三浦織絵です。
自分のやりたいことを模索するうちに「Webデザイン」という仕事に出会い、志すようになりました。今はオンラインスクールで基礎からwebデザインについて学んでいます。 今後はさらにWeb制作の技術や知識を身につけ、クライアント・ターゲット両方の要望を実現できるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングを勉強中です。これからは使いやすく、見やすいサイトを作るためにデザイン知識とそれを表現するためのPhotoshopやIllustratorといったツールの活用方法を徐々に身につけていきたいです。

Illust

イラストを描くのが好きで、サークルのTシャツデザイン、会報の表紙イラストや新歓用のパンフレットを制作しました。今後はデジタルでの制作活動をしていきたいです。

hobby

趣味は歌を歌うこと・聞くことで、合唱サークルで定期演奏会に向け毎週楽しく練習しています。また半年前から好きになったKPOPの影響で韓国語の勉強を始めました。
